# GATE 2027 Extension Icons

The extension uses text-based icons. To add proper PNG icons:

1. Create icon PNG files at these sizes:
   - icons/icon16.png (16×16)
   - icons/icon32.png (32×32)  
   - icons/icon48.png (48×48)
   - icons/icon128.png (128×128)

2. You can generate them at https://favicon.io or use any icon editor.

3. Suggested design: Purple/indigo graduation cap on dark background.

If icons are missing, Chrome will use a default icon — the extension will still work.
